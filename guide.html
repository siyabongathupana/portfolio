User Guide

Everything you need to know about using Your Portfolio

Getting Started

1 Create an account
Go to Login page → "Create new account". Enter your email, a strong passphrase (min 8 characters), and your GitHub Personal Access Token (requires repo scope). You'll receive a confirmation email.
2 Log in
Use your email and passphrase. Your session lasts 24 hours. You'll be redirected to the Admin Dashboard.
3 Your data is private
All your projects, certificates, and timesheet entries are stored in your own GitHub folder. Only you (and admin) can see them.

Managing Projects

Add a new project
In the Admin Dashboard → Projects tab → "Add New Project". Fill in details: title, controller type, cabinets, I/O counts, dates, team members, description. You can drag & drop images – they are compressed and uploaded to GitHub automatically.
Edit / Delete
Each project has Edit and Delete buttons. Deleting also removes uploaded images from GitHub.
PDF Report
On the public portfolio pages, click "PDF Report" to generate a beautiful summary with I/O charts and project images.

Certificates

Add a certificate
In Admin → Certificates → "Add Certificate". Provide title, issuer, date, a link (PDF or Google Drive), and an optional thumbnail. Thumbnails are compressed automatically.
Public view
Certificates appear on the Certificates page. Visitors can click "View / Download" to open the linked document.

Timesheet

Log your time
Use the quick log form: select date, start/end (click "Now" for current time), choose project & category, mark billable, add notes. Hours are calculated automatically.
Edit / Delete entries
Each entry has a pencil icon (edit) and trash icon (delete). Edits open a modal to modify any field.
Filters & charts
Filter by today, week, month, or all time. Also filter by project and category. Three pie charts show breakdowns.
Export & Print
Export filtered data to Excel (.xlsx) or generate a PDF report with a custom date range. The print button uses print‑optimised CSS.
Overtime tracking
Daily progress bar turns red if you exceed 8 hours. The summary card shows overtime for the selected period.

Admin Features Admin only

User management
In Admin → Users tab, you can block/unblock users, delete user accounts (and all their data), and view their activity logs.
Backup & Restore
Download the latest daily backup (tar.gz) from GitHub, or restore from a previously downloaded ZIP backup.
Activity logs
Download your own logs as a colour‑coded PDF, or (as admin) download any user's logs or all users' logs in a ZIP.

Security & Support

Data encryption
Your GitHub token is encrypted with AES‑GCM using your passphrase. Never share your passphrase.
Session security
Sessions expire after 24 hours. Log out manually from the footer or the admin dashboard.
